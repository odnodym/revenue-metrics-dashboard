# Revenue Metrics Dashboard

## Project Overview
This project analyzes revenue and user subscription metrics using SQL and Tableau Public.

## Metrics
- MRR
- Paid Users
- ARPPU
- New Paid Users
- New MRR
- Churned Users
- Churn Rate
- Churned Revenue
- Revenue Churn Rate
- Expansion MRR
- Contraction MRR
- LT
- LTV

## Tools
- SQL
- PostgreSQL
- Tableau Public
- DBeaver
